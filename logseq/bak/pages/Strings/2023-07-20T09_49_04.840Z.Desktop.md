- Зрізи (**Slices**):
	- Увага, з огляду на те, що початок **зрізу** завжди включається, кінець - завжди виключено.
	  Тобто, **s[:i] + s[i:]** завжди дорівнює **s**
	- Code
		- ```python
		  word = 'Python'
		  word[:2] + word[2:] == word
		  ```
